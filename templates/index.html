How It Works

Three independent AI modules analyse every document and their results are fused into a single authenticity score.

Visual Analysis 50%

Error Level Analysis (ELA) re-compresses the document and maps pixel-level inconsistencies that reveal splicing, copy-paste edits, and JPEG manipulation.

Text Extraction 30%

EasyOCR reads names, dates, IDs, and labels from the document. Low-confidence text blocks signal font replacement, text alterations, or unusual character patterns.

Forensic Detection 20%

ORB keypoint matching with RANSAC homography identifies regions copied and pasted within the same document — a common technique in certificate fraud.

Fusion Formula Final Score = 0.5 × Visual + 0.3 × OCR + 0.2 × Forensic Score ≥ 0.65 → Genuine | 0.40–0.64 → Suspicious | < 0.40 → Forged
